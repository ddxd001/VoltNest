# 视频切换优化说明

## 🎯 功能说明

### 视频播放模式
**单视频循环播放模式**：
- 每个视频独立循环播放
- 只在按下 C/V 键时才切换到其他视频
- 不按键时保持当前视频持续循环

### 核心特性

1. **单视频循环**
   - VLC 播放单个视频并循环
   - 不会自动切换到下一个视频
   - 保持当前表情持续显示

2. **手动切换**
   - 按 C 键：切换到下一个视频
   - 按 V 键：切换到上一个视频
   - 切换后新视频开始循环播放

3. **快速切换**
   - 停止当前 VLC 进程
   - 启动新视频的 VLC 进程
   - 切换时间约 1-2 秒（包含黑屏过渡）

## 🔧 技术实现

### 单视频循环播放
```python
def _start_vlc_single_video(self, video_path):
    """启动VLC播放单个视频（循环播放）"""
    vlc_cmd = [
        "cvlc",
        "--fullscreen",
        "--loop",  # 循环播放当前视频
        "--no-video-title-show",
        "--no-osd",
        "--quiet",
        video_path
    ]
    
    self.vlc_process = subprocess.Popen(
        vlc_cmd,
        stdout=subprocess.DEVNULL,
        stderr=subprocess.DEVNULL,
        env={**os.environ, 'DISPLAY': os.environ.get('DISPLAY', ':0')}
    )
```

### 视频切换逻辑
```python
def switch_to_next(self):
    """切换到下一个视频"""
    with self.lock:
        self.current_video_index = (self.current_video_index + 1) % len(self.video_files)
        self.switch_event.set()  # 触发切换事件
```

### 主循环
```python
def run(self):
    """视频播放主循环（单视频循环模式）"""
    while not self.stop_event.is_set():
        # 启动当前视频循环播放
        self._start_vlc_single_video(video_path)
        
        # 等待切换事件
        while not self.stop_event.is_set() and not self.switch_event.is_set():
            time.sleep(0.1)
        
        # 停止当前视频，准备切换
        self._stop_current_vlc()
        self.switch_event.clear()
```

## 🐛 故障排查

### 1. 切换无响应
**可能原因**：VLC 进程未正常启动或视频文件路径错误

**解决方法**：
```bash
# 检查 VLC 进程
ps aux | grep vlc

# 重启 VLC
pkill vlc
# 然后重新启动遥操作程序
```

### 2. 视频播放异常
**可能原因**：视频文件路径错误或文件损坏

**解决方法**：
```bash
# 检查视频文件
ls /home/ubuntu/lerobot_alohamini/face_video/*.mp4

# 检查文件权限
chmod 644 /home/ubuntu/lerobot_alohamini/face_video/*.mp4
```

### 3. 视频不循环播放
**可能原因**：VLC 参数错误

**解决方法**：
- 检查 VLC 启动参数中是否包含 `--loop`
- 重启树莓派端程序

## 📝 注意事项

1. **视频格式**
   - 推荐使用 MP4 格式
   - 编码：H.264
   - 分辨率：建议不超过 1920x1080

2. **视频数量**
   - 支持任意数量的视频
   - 自动循环播放
   - 按文件名排序

3. **性能优化**
   - 视频文件应存储在本地磁盘
   - 避免使用网络共享文件
   - 定期清理临时文件
