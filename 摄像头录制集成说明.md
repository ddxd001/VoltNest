# 📹 摄像头录制集成说明

## 🎯 功能概述

`start_pi.sh` 现在支持在启动遥操作服务时**自动启动摄像头录制**，实现一键启动、同步录制的功能。

---

## ⚙️ 配置方法

### 开启摄像头录制

编辑 `start_pi.sh` 文件，修改配置项：

```bash
# ============ 配置选项 ============ #
# 自动更新开关：true=启用自动更新，false=禁用自动更新
AUTO_UPDATE=true

# 摄像头录制开关：true=启用录制，false=禁用录制
ENABLE_CAMERA_RECORDING=true    # 改为 true
# ================================= #
```

### 关闭摄像头录制

```bash
ENABLE_CAMERA_RECORDING=false   # 改为 false（默认）
```

---

## 🚀 使用方法

### 方式 1：启用录制（推荐）

```bash
# 1. 编辑 start_pi.sh，设置 ENABLE_CAMERA_RECORDING=true

# 2. 启动服务（自动开始录制）
./start_pi.sh
```

**输出示例**：
```
================================================
  启动摄像头录制
================================================

📹 启动摄像头录制（后台运行）...
✅ 摄像头录制已启动 (PID: 12345)
   录制日志: camera_recorder/recorder.log

================================================
  启动从臂控制服务
================================================

配置信息：
  - 机械臂类型: so-arm-5dof
  - 监听端口: 5555 (命令), 5556 (观测)
  - 控制频率: 30 Hz
  - 看门狗超时: 1500 ms
  - 摄像头录制: 已启用 (PID: 12345)

⚠️  注意事项：
  1. 确保从臂已上电
  2. 确保串口连接正常
  3. 等待 Mac 端连接后开始遥操作
  4. 摄像头正在录制，停止服务时录制也会自动停止

按 Ctrl+C 停止服务
```

### 方式 2：不启用录制

```bash
# 1. 编辑 start_pi.sh，设置 ENABLE_CAMERA_RECORDING=false（默认）

# 2. 启动服务（不录制）
./start_pi.sh
```

## 🔄 工作流程

### 启用录制时的流程

```
1. 运行 start_pi.sh
   ↓
2. 自动更新代码（如果启用）
   ↓
3. 激活虚拟环境
   ↓
4. 检查串口设备
   ↓
5. 启动摄像头录制（后台）
   ↓
6. 启动从臂控制服务
   ↓
7. 按 Ctrl+C 停止
   ↓
8. 自动停止摄像头录制
   ↓
9. 自动停止从臂服务
```

### 禁用录制时的流程

```
1. 运行 start_pi.sh
   ↓
2. 自动更新代码（如果启用）
   ↓
3. 激活虚拟环境
   ↓
4. 检查串口设备
   ↓
5. 启动从臂控制服务（跳过录制）
   ↓
6. 按 Ctrl+C 停止
   ↓
7. 自动停止从臂服务
```

## 🐛 故障排查

### 问题 1：录制未启动

**症状**：启动时显示"摄像头录制启动失败"

**解决方法**：
```bash
# 查看录制日志
cat ~/lerobot_alohamini/camera_recorder/recorder.log

# 测试摄像头
cd ~/lerobot_alohamini/camera_recorder
python scripts/test_camera.py

# 检查摄像头设备
ls -l /dev/video*
```

### 问题 2：录制模块未找到

**症状**：显示"未找到摄像头录制模块"

**解决方法**：
```bash
# 拉取最新代码
cd ~/lerobot_alohamini
git pull origin main

# 检查模块是否存在
ls -la camera_recorder/
```

### 问题 3：录制文件过大

**解决方法**：
```bash
# 降低分辨率（编辑 camera_recorder/config.yaml）
camera:
  width: 640
  height: 480
  fps: 30
```

## 📝 注意事项

### 1. **存储空间**
- 720p @ 30fps ≈ 100-150 MB/分钟
- 定期清理旧录像：`rm ~/lerobot_alohamini/camera_recorder/recordings/*.mp4`

### 2. **性能影响**
- 录制在后台运行，不影响遥操作性能
- 如果遇到性能问题，可以降低录制分辨率

### 3. **自动停止**
- 按 Ctrl+C 停止服务时，录制会自动停止
- 录制文件会自动保存，不会丢失

### 4. **日志查看**
- 录制日志：`camera_recorder/recorder.log`
- 可以实时查看：`tail -f camera_recorder/recorder.log`
