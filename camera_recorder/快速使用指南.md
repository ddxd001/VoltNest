# 📹 摄像头录制模块 - 快速使用指南

## 🚀 5分钟快速上手

### 步骤 1：安装依赖

```bash
cd ~/lerobot_alohamini/camera_recorder
pip install -r requirements.txt
```

### 步骤 2：测试摄像头

```bash
python scripts/test_camera.py
```

**输出示例**：
```
正在扫描可用摄像头...
==================================================

测试摄像头 0...
  ✅ 摄像头 0 可用
     分辨率: 1280x720
     帧率: 30 fps
     设备路径: /dev/video0

==================================================
扫描结果:
==================================================

找到 1 个可用摄像头:
  - 摄像头 0 (/dev/video0)

💡 提示:
   在 config.yaml 中设置 device_id: 0 来使用第一个摄像头
```

### 步骤 3：修改配置（可选）

编辑 `config.yaml`：

```yaml
camera:
  device_id: 0        # 使用测试脚本找到的摄像头 ID
  width: 1280         # 分辨率宽度
  height: 720         # 分辨率高度
  fps: 30             # 帧率
```

### 步骤 4：开始录制

```bash
./scripts/start_recorder.sh
```

**输出示例**：
```
================================================
  Camera Recorder
  启动即录制，Ctrl+C 停止
================================================

[INFO] Initializing camera device 0...
[INFO] Camera initialized: 1280x720 @ 30fps
[INFO] Recording started: ./recordings/recording_20260314_110730.mp4
[INFO] Codec: mp4v, Resolution: 1280x720, FPS: 30
[INFO] Press Ctrl+C to stop recording
```

### 步骤 5：停止录制

按 `Ctrl+C` 或在另一个终端运行：

```bash
./scripts/stop_recorder.sh
```

**输出示例**：
```
==================================================
Recording stopped
Output file: ./recordings/recording_20260314_110730.mp4
Total frames: 3000
Duration: 100.0s (1.7min)
File size: 125.3 MB
==================================================
```

## 📊 与遥操作同时使用

### 方案 1：两个终端（推荐）

**终端 1 - 启动录制**：
```bash
ssh ubuntu@192.168.3.33
cd ~/lerobot_alohamini/camera_recorder
./scripts/start_recorder.sh
```

**终端 2 - 启动遥操作**：
```bash
ssh ubuntu@192.168.3.33
cd ~/lerobot_alohamini
./start_pi.sh
```

### 方案 2：后台运行

**启动录制（后台）**：
```bash
cd ~/lerobot_alohamini/camera_recorder
nohup ./scripts/start_recorder.sh > recorder.log 2>&1 &
```

**查看录制状态**：
```bash
tail -f ~/lerobot_alohamini/camera_recorder/recorder.log
```

**停止录制**：
```bash
./scripts/stop_recorder.sh
```

## 📁 录制文件管理

### 查看录制文件

```bash
ls -lh ~/lerobot_alohamini/camera_recorder/recordings/
```

### 下载到 Mac

```bash
# 下载所有录制文件
scp ubuntu@192.168.3.33:~/lerobot_alohamini/camera_recorder/recordings/*.mp4 ~/Downloads/

# 下载指定文件
scp ubuntu@192.168.3.33:~/lerobot_alohamini/camera_recorder/recordings/recording_20260314_110730.mp4 ~/Downloads/
```

### 清理旧文件

```bash
# 删除所有录制文件
rm ~/lerobot_alohamini/camera_recorder/recordings/*.mp4

# 删除 7 天前的文件
find ~/lerobot_alohamini/camera_recorder/recordings/ -name "*.mp4" -mtime +7 -delete
```

## 🐛 常见问题

### 问题 1：找不到摄像头

```bash
# 检查摄像头设备
ls -l /dev/video*

# 测试摄像头
python scripts/test_camera.py
```

### 问题 2：权限不足

```bash
# 添加用户到 video 组
sudo usermod -a -G video $USER

# 重新登录生效
```

### 问题 3：录制文件无法播放

```bash
# 转换格式
ffmpeg -i recording.mp4 -c:v libx264 recording_converted.mp4
```

### 问题 4：磁盘空间不足

```bash
# 检查磁盘空间
df -h

# 清理旧录像
rm ~/lerobot_alohamini/camera_recorder/recordings/*.mp4
```

### 1. 自动命名

录制文件自动按时间戳命名，无需手动指定：
- `recording_20260314_110730.mp4`（2026年3月14日 11:07:30）

### 2. 多摄像头录制

如果有多个摄像头，可以启动多个录制实例：

```bash
# 修改 config.yaml 的 device_id 为不同的值
# 然后在不同终端启动
```

### 3. 定时录制

使用 `timeout` 命令限制录制时长：

```bash
# 录制 5 分钟后自动停止
timeout 300 ./scripts/start_recorder.sh
```

### 4. 监控存储空间

```bash
# 实时监控磁盘使用
watch -n 5 df -h
```
